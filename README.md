# 💰 Expense Tracker App
A modern and simple **Expense Tracker Application** built using **Vite + TypeScript** and packaged for Android using **Capacitor**.
This project helps users manage daily expenses, track transactions, and maintain spending records efficiently.
---
## 🚀 Features
✅ Add new income and expense transactions  
✅ View all recent transactions clearly  
✅ Categorized expense tracking  
✅ Clean and responsive UI  
✅ Android APK support using Capacitor  
✅ Fast performance with Vite + TypeScript  
---
## 🛠️ Tech Stack
- **Frontend:** Vite + TypeScript  
- **Mobile Packaging:** Capacitor  
- **Platform:** Android  
- **Build Tools:** Node.js + npm

⚡ Getting Started (For Any User)
Follow these steps to run the project locally:
✅ 1. Clone the Repository
git clone https://github.com/bot041/Expense-Tracker-App.git
✅ 2. Navigate Into the Project Folder
cd Expense-Tracker-App
✅ 3. Install Dependencies
npm install
✅ 4. Start Development Server
npm run dev
The application will run at:
http://localhost:5173

📦 Build the Project for Production
npm run build
📱 Run as Android App (APK Build)
Capacitor allows packaging your web app into Android.
✅ Step 1: Sync Capacitor Project
npx cap sync
✅ Step 2: Open Android Studio
npx cap open android
✅ Step 3: Build APK in Android Studio
Inside Android Studio, go to:
Build → Build Bundle(s)/APK(s) → Build APK(s)

Your APK will be generated successfully.
